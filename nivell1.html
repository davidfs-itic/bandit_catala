Objectiu del nivell

La contrasenya per al següent nivell està emmagatzemada en un fitxer anomenat readme ubicat al directori principal. Utilitza aquesta contrasenya per iniciar sessió a bandit1 mitjançant SSH. Sempre que trobis una contrasenya per a un nivell, usa SSH (al port 2220) per accedir a aquest nivell i continuar el joc.

Comandaments que podrien ser necessaris

  • ls
  • cd
  • cat
  • file
  • du
  • find

Consell

Crea un fitxer local per a notes i contrasenyes. Les contrasenyes no es desen automàticament. Si no les guardes, hauràs de començar des de bandit0. Addicionalment, es recomana prendre notes detallades per a nivells complexos, ja que les contrasenyes poden canviar ocasionalment.
